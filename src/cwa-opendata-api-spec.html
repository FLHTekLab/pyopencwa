array[string]

string
(query)

startTime
array[string]
(query)

timeFrom
string($date-time)
(query)

時間區段，篩選需要之時間區段，時間從「timeFrom」 開始篩選，直到內容之最後時間，並可與參數 「timeTo」 合併使用，格式為「yyyy-MM-ddThh:mm:ss」，若使用參數「startTime」，則參數「timeFrom」的篩選資料則會失去作用， 預設為全部回傳

timeTo
string($date-time)
(query)

時間區段，根據內容可篩選需要之時間區段，時間從內容之最初時間開始篩選，直到 「timeTo」，並可與參數 「timeFrom」 合併使用，格式為「yyyy-MM-ddThh:mm:ss」，若使用「startTime」，則參數「timeTo」的篩選資料則會失效，預設為全部回傳
