format
string
(query)

Available values : JSON, XML

AreaName
array[string]
(query)

StationName
array[string]
(query)

測站名稱，預設為全部回傳

sort
array[string]
(query)

「OriginTime」為針對「OriginTime」與「ReportNo」做升冪排序，預設「OriginTime」與「ReportNo」為降冪排序

Available values : OriginTime

OriginTime
array[string]
(query)

時間因子，格式為「yyyy-MM-ddThh:mm:ss」，預設為全部回傳

timeFrom
string($date-time)
(query)

時間區段，篩選需要之時間區段，時間從「timeFrom」開始篩選，直到內容之最後時間，並可與參數 「timeTo」 合併使用，格式為「yyyy-MM-ddThh:mm:ss」，若使用參數「dataTime」，則參數「timeFrom」的篩選資料則會失效 ，預設為全部回傳

timeTo
string($date-time)
(query)

時間區段，篩選需要之時間區段，時間從內容之最初時間開始篩選，直到「timeTo」，並可與參數 timeFrom」 合併使用，格式為「yyyy-MM-ddThh:mm:ss」，若使用「dataTime」，則參數「timeTo」的篩選資料則會失效，預設為全部回傳
